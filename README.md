# IPL First Innings Score Prediction using Machine Learning

## Overview
This project predicts the first innings score in IPL matches using machine learning models trained on historical match data.

## Problem Statement
The goal is to predict the final first innings score based on match conditions such as team, overs, runs, and wickets.

## Features Used
- Batting Team
- Bowling Team
- Overs
- Runs
- Wickets
- Runs in Last 5 Overs
- Wickets in Last 5 Overs

## Dataset
The dataset contains ball-by-ball IPL data with relevant match features required for prediction.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Building
6. Model Evaluation
7. Prediction

## Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- AdaBoost Regressor

## Model Performance
- R2 Score: 0.97
- Low Mean Absolute Error
- Low Root Mean Squared Error

## Example Input
Batting Team: Mumbai Indians
Bowling Team: Chennai Super Kings
Overs: 15.3
Runs: 120
Wickets: 3

## Output
Predicted Score: 175-185
